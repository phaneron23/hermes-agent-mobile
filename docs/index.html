🚀 Two modes. One cockpit.

Hermes Agent Mobile is designed for both portable autonomy and remote command. Local when you want sovereignty. Remote when you want big-machine power.

📱 Fully Local Mode

Run the agent stack directly on Android. Proot Linux runtime, Node.js agent console, SSH gateway, local dashboard, and project tooling — all from the phone.

🌐 Remote Hermes Mode

Connect to an existing Hermes instance. Use your phone as the command center for a desktop, server, cloud VM, or private homelab agent.

🔥 What you get

💬 Agent Chat

Keep conversations, tools, files, and model switching close at hand.

🐧 Android Linux Runtime

Run a bundled Linux environment locally without rooting the device.

🔌 SSH Gateway

Forward ports, open a terminal, debug the runtime, and go deeper.

📁 Project Navigation

Inspect repos, files, and workspace state from your phone.

🧠 Model Routing

Use the right model for each job and keep provider choice flexible.

⚙️ Task Control

Launch, monitor, resume, and babysit long-running agent work.
